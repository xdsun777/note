Deepin Linux显卡设置

2022-05-25 2 min read # Linux

注意：\color{blue}{注意：}注意：本文章适用于intel集成显卡与nvidia独显的机器

1.安装nvidia闭源驱动（一般在安装系统时，提供了选择）

#安装nvidia闭源驱动
sudo apt-get install -y nvidia-detect
nvidia-detect | awk 'match($0, /nvidia-.*/, a) {print a[0]}' | xargs sudo apt-get -y install

2.nvidia相关工具

sudo apt install nvidia-smi nvidia-settings

nvidia-settings 是设置工具，提供GUI
nvidia-smi 是显卡监控工具，提供显卡使用信息

watch -n 1 nvidia-smi

3.intel集成显卡监控工具

sudo apt install intel-gpu-tools

# 使用
intel-gpu-overlay # GUI实时显示gpu信息

intel_gpu_top # 终端显示GPU信息

intel_gpu_frequency # 简短显示GPU使用信息

intel_gpu_time cmd # 显示cpu和gpu运行程序时的使用信息

4.切换intel核显与nvidia独显

安装任务栏显卡驱动切换插件

wget -t 3 -T 15 https://github.com/zty199/dde-dock-switch_graphics_card/releases/download/v1.8.4-1/dde-dock-graphics-plugin_1.8.4-1_amd64.deb
sudo apt-get install -y ./dde-dock-graphics-plugin_1.8.4-1_amd64.deb
