github ssh免密与token登录配置

2022-05-29 1 min read # git

今天git push突然报错，可能是密钥过期了吧，于是重新配置了一下

remote: Support for password authentication was removed on August 13, 2021. Please use a personal access token instead.
remote: Please see https://github.blog/2020-12-15-token-authentication-requirements-for-git-operations/ for more information.
fatal: Authentication failed for 'https://github.com/CharmGo/sun.git/'

生成密钥

ssh-keygen -t rsa -b 4096 -C "charmgo@qq.com"

cat ~/.ssh/生成的密钥.pub

将cat打印的值复制到settings>SSH and GPG keys>New SSh key>
