ffmpeg相关使用

2022-05-27 2 min read # ffmpeg

ffmpeg是用来记录、转换数字音频、视频，并能将其转化为流的开源软件。

视频相关

flv转mp4

ffmpeg -i j.flv -c copy j.mp4

模糊视频水印

ffmpeg -i ./1.mp4 -filter_complex "delogo=x=30:y=210:w=500:h=140:" 4.mp4

合并mp4格式视频

ffmpeg -i ./4.mp4 -vcodec copy -acodec copy -vbsf h264_mp4toannexb 1.ts
ffmpeg -i ./5.mp4 -vcodec copy -acodec copy -vbsf h264_mp4toannexb 2.ts

ffmpeg -i "concat:1.ts|2.ts" -acodec copy -vcodec copy -absf aac_adtstoasc output.mp4

截取视频

-ss 起始位置
-t 结束位置

ffmpeg -ss 00:00:03 -t 00:02:30 -i ./1.mp4 -vcodec copy -acodec copy out.mp4

图片相关

转换图片格式

ffmpeg -i 初始文件 输出文件

压缩图片

ffmpeg -i image_source -q compress_level out_source
#compress_level：压缩等级为正整数，数值越大压缩等级越高

图片合成视频

ffmpeg -i ./convert/mn3_%6d.png -c:v libx264 -vf "fps=25,format=yuv420p" out3.mp4

音频相关

mp3转换成wav

ffmpeg -i input.mp3 -acodec pcm_u8 -ar 22050 out.wav
